# 👻 Pac-Man

迷路の中でドットを食べながら幽霊から逃げる定番アーケードゲーム。

## 遊び方
- **↑↓←→ / WASD** — パックマンを移動
- クリックまたはキー入力でゲームスタート
- スマホはスワイプ操作に対応

## ルール
1. 迷路内のドット（●）をすべて食べるとステージクリア
2. 幽霊に触れるとライフが1つ減る（3ライフ制）
3. パワーエサ（大きいドット）を食べると幽霊が青くなり逆に食べられる

## 得点
| アイテム | 点数 |
|---------|------|
| ドット | 10 |
| パワーエサ | 50 |
| ゴースト（1体目） | 200 |
| ゴースト（連続） | 400, 800, 1600 |
| フルーツ | 100〜3000（レベルで変化） |

## 特徴
- 4体の幽霊がそれぞれ異なるAIで追跡
  - **Blinky（赤）**: パックマンを直接追跡
  - **Pinky（ピンク）**: パックマンの4マス先を狙う
  - **Inky（水色）**: Blinkyとの位置関係で挟み撃ち
  - **Clyde（オレンジ）**: 近いと逃げ、遠いと追跡
- パワーアップ状態でのゴースト点滅演出
- レベルアップで幽霊が速くなる
- 左右のトンネル貫通
- フルーツボーナス（🍒🍓🍊🍎🍇🔔🔑）
- ハイスコア保存（localStorage）
- サウンドエフェクト（Web Audio API）
- 「READY!」演出

## 技術スタック
HTML5 Canvas / Vanilla JavaScript / Web Audio API
